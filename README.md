# Bili Identity [WIP]

## 项目简介

**Bili Identity** 旨在为各类网站、社区、个人站点提供一个简易、可自托管的 Bilibili 账号身份认证兼容层。
本项目通过 B 站私信验证码的方式，间接验证用户 B 站身份，并对外以 OpenID Connect 接口提供标准认证服务。

**适用场景**：

- Bilibili 官方未提供 OAuth2/OpenID 的站点认证API访问权限
- 需要以低成本、开源自托管方式接入 B 站账号登录的第三方网站
